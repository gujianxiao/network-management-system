About System

NDN Web System is a prototype system,it belongs to our project named GlobeSen,or we can call it SenBrowser.The destination of this syetem is to show the info that we get from the sink.

To use this system,you may have to install some software on your compputer.First,the Ndnx protocal.Ndnx is a prototal that provide a base communication protocal of this system.You can see more information and get to know how to install Ndnx on your computere via http://named-data.net/ .Second,you should make sure your OS is Linux,because our system call some function belong to Linux,and Ndnx should run on Linux OS.Finally,just personal sugguestion:please use Firefox,Chrome or Opera etc,IE may cause some strange problem.
